# Spoiler

Spoiler widget for flutter.

## HowTo:
```dart
child: Spoiler(
            openCurve: Curves.elasticOut,
            closeCurve: Curves.elasticIn,
            header: Text("Tools", style: TextStyle(color: Colors.white)),
            child: GameControl()
```
